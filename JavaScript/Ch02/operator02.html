연산자(operator)

3. 비교(관계)연산자 : >, >=, <, <=, ==, !=, ===, !== - >, >=, <, <=, ==, != : 값만 비교하여 true 혹은 false로 출력 - ===, !== : 자바스크립트에서 추가된 연산자, 값과 타입을 한꺼번에 비교하여 true 또는 false로 출력 - 우선순위 : (>, >=, <, <=) 연산자가 (==, !=) 보다 우선임 - 우선순위 : (==, !=) 연산자가 (===, !==)보다 우선임 4. 논리연산자 : &&(논리 AND, 논리곱), ||(논리 OR, 논리합), !(논리 NOT, 논리 부정) - 논리연산자의 결과는 논리값(true, false 둘 중의 하나) - 비교연산자가 논리연산자보다 우선임, 비교 > 논리 - 논리AND가 논리OR보다 우선임. 5. 대입연산자 : = 복합대입연산자(할당연산자, assignment) : +=, -=, *=, /=, %=
